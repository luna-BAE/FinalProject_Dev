프로젝트 소개
------------
> 코워킹 스페이스 운영/관리와 멤버 커뮤니티를 위한 웹 사이트 구축
> - 건물과 멤버 관리를 하나의 사이트에서 관리
> - 편리한 예약 시스템과 커뮤니티 제공

담당 역할
-----------
> - 데이터베이스 설계
> - 멤버 커뮤니티 & UI   
> - 게시판, 페이징, 댓글   
> - 회의실 예약 시스템   
> - 예약결제 / 결제취소   
> - 스케쥴러(메일 발송)
